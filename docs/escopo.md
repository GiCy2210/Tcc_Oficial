# Escopo do Projeto

## Título
Sistema Inteligente de Apoio à Correção de Redações no Modelo ENEM

## Problema
Estudantes que se preparam para o ENEM têm dificuldade em obter feedback rápido, detalhado e personalizado para suas redações, limitando sua evolução.

## Solução
Plataforma web que utiliza IA generativa (Google Gemini) para analisar redações, apontar erros e sugerir plano de estudos personalizado.

## Funcionalidades do MVP
1. **Corretor automático** - Avalia nas 5 competências ENEM
2. **Apontador de erros** - Gramática, coesão e argumentação
3. **Plano de estudos** - Recomendações personalizadas

## Fora do Escopo (por enquanto)
- Chat tutor interativo
- Sistema de login/usuários
- Geração automática de temas
- Fine-tuning de modelo próprio
- App mobile

## Tecnologias
- Python + Streamlit
- Google Gemini API
- spaCy + LangChain
- FAISS (RAG futuro)
