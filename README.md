# 📝 TCC: Sistema Inteligente de Apoio à Correção de Redações ENEM

## 🎯 Sobre o Projeto
Sistema baseado em IA para auxiliar estudantes na preparação para a redação do ENEM, oferecendo:
- Correção automática nas 5 competências
- Identificação de erros gramaticais e argumentativos
- Plano de estudos personalizado
